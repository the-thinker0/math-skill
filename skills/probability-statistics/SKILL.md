# 🎲 概率与统计

> "量化不确定性，从数据中提取规律，做出理性的推断和决策。"
> "Quantifying uncertainty, extracting patterns from data, and making rational inferences and decisions."
>
> —— 概率论、数理统计
> —— Probability Theory, Mathematical Statistics

## 核心原则 / Core Principle

**世界充满不确定性。理性决策不是追求确定性（那几乎不存在），而是在不确定性中做出期望值最优的选择。概率是量化的信念，数据是量化的证据。**

**The world is full of uncertainty. Rational decision-making is not about pursuing certainty (which almost never exists), but about making choices with the best expected value under uncertainty. Probability is quantified belief, data is quantified evidence.**

概率论的根基是 Kolmogorov 公理体系——概率空间 (Ω, 𝔽, P) 由三元组定义：

- **非负性公理**：P(A) ≥ 0，对 𝔽 中任意事件 A 成立
- **规范性公理**：P(Ω) = 1，必然事件概率为 1
- **可加性公理**：若 A₁, A₂, … 为 𝔽 中互不相交的可数事件族，则 P(∪ᵢ Aᵢ) = Σᵢ P(Aᵢ)（可数可加性）

三条公理推导出全部概率定理——互补律 P(Aᶜ) = 1 − P(A)、包含律 P(A∪B) = P(A) + P(B) − P(A∩B)、单调性 A ⊆ B ⇒ P(A) ≤ P(B)、全概率公式 P(B) = Σᵢ P(B|Aᵢ)P(Aᵢ)。Kolmogorov 公理是概率论的"欧几里得公理"，一切统计方法的合法性最终回溯于此。

概率思维的核心概念：
- **概率是信念的度量**：先验概率 → 新证据 → 后验概率（贝叶斯更新）
- **大数定律**：单次观察不可靠，大量观察才揭示真实规律；n → ∞ 时样本频率收敛于理论概率
- **期望值**：理性决策的依据是期望值 E[X] = Σx·P(x)，不是最可能的结果
- **方差与风险**：期望值相同的情况下，Var(X) = E[(X−μ)²] 代表波动与风险
- **偏差-方差权衡**：模型复杂度 ↑ ⇒ 偏差 ↓、方差 ↑；总误差 = 偏差² + 方差 + 不可约噪声

> 详细数学依据见 `original-texts.md`

## 不适用场景 / When NOT to Use

- **确定性问题**（已知答案或可通过演绎推理得到）——不需要概率工具
- **样本量极小（n < 5）且无法增加**——统计推断需要足够的样本，此时应报告数据本身而非推断
- **纯粹的定性描述需求**（如"解释一下这个概念"）——不需要数据分析
- **因果机制已明确且无需概率建模**——直接使用确定性因果链即可
- **数据完全缺失或严重损坏**——无法进行任何有意义的统计操作

## 何时使用 / When to Use

- 阅读论文时，审查其统计方法是否正确、结论是否显著
- 实验数据分析、结果显著性检验、效应量评估
- 解读新闻或报告中的统计数据，识别统计谬误
- 评估风险（投资风险、健康风险、决策风险）
- 避免认知偏差（幸存者偏差、确认偏差、基础率忽视）
- 在日常生活中做出概率性决策
- 回归建模与预测（线性回归、逻辑回归、模型选择）
- 实验设计与功效分析（随机化、区组化、因子设计）
- 因果推断（从观测数据识别因果效应，DAG + 后门准则）
- 拟合优度检验与方差比较（χ² 检验、F 检验）

## 方法流程 / Method

### 第一步：定义随机现象 / Define the Random Phenomenon

明确：
- **随机变量 X**是什么？（你要测量的量）
- **样本空间 Ω**是什么？（所有可能结果的集合）
- **事件域 𝔽**：Ω 的哪些子集可被赋予概率？（满足 σ-代数条件：含 Ω、对补封闭、对可数并封闭）
- 随机变量是**离散的**还是**连续的**？
- **概率测度 P**：Ω → [0,1] 的映射，满足 Kolmogorov 三公理

### 第二步：选择概率模型 / Choose the Probability Model

根据现象的性质选择合适的分布：

| 分布 / Distribution | 适用场景 / Use Case | 关键参数 / Key Parameters |
|---|---|---|
| 二项分布 Binomial | N 次独立伯努利试验中成功次数 | n（试验数），p（成功概率） |
| 泊松分布 Poisson | 单位时间/空间内稀有事件发生次数 | λ（平均发生率） |
| 正态分布 Normal | 大量独立小因素叠加（中心极限定理） | μ（均值），σ²（方差） |
| 指数分布 Exponential | 等待时间、无记忆性 P(X>t+s|X>s)=P(X>t) | λ（速率） |
| 贝叶斯模型 Bayesian | 先验 + 似然 → 后验，逐步更新信念 | 先验参数，似然函数 |
| t-分布 Student's t | 小样本均值推断（Student 1908），尾部比正态厚 | df = n − 1（自由度） |
| χ²-分布 Chi-squared | 拟合优度检验、列联表分析、方差估计 | df（自由度） |
| F-分布 F | 比较两组方差、ANOVA 方差分析 | df₁, df₂（两个自由度） |
| 均匀分布 Uniform | 等概率事件、随机数生成、无先验信息时的默认 | a, b（区间端点） |

选择原则：数据类型（离散/连续）、样本量（大样本趋正态，小样本用 t）、物理机制（泊松对应稀有独立事件、指数对应等待时间）。

### 第三步：收集数据 / Collect Data

- 样本量是否足够？（小样本的推断不可靠；功效分析确定最小 n）
- 样本是否有偏？（选择偏差、幸存者偏差、自愿响应偏差）
- 数据质量如何？（测量误差、缺失值、异常值检测）
- 收集方式是否随机化？（随机抽样 vs 方便抽样）
- 是否存在区组结构？（区组化可减少混杂变异）

### 第四步：统计推断 / Statistical Inference

- **点估计**：用样本统计量估计总体参数（样本均值 x̄ → μ）
- **最大似然估计 MLE**：选择使似然函数 L(θ; x) = f(x|θ) 最大的 θ̂ = argmax_θ L(θ; x)；大样本下 MLE 渐近正态、渐近有效
- **置信区间**：以指定置信水平（如 95%）给出参数的区间估计；频率学派解释：重复抽样时 95% 的区间包含真实参数
- **假设检验**：H₀ vs H₁，p-value = P(观察到此数据或更极端 | H₀)；p < α 则拒绝 H₀
- **贝叶斯更新**：后验 P(θ|D) = P(D|θ)·P(θ)/P(D)；先验 → 似然 → 后验，后验再遇新数据成为新先验

### 第五步：回归建模 / Regression Modeling

- **线性回归**：Y = β₀ + β₁X + ε，ε ~ N(0, σ²)；最小二乘估计 β̂ = (XᵀX)⁻¹XᵀY；假设：线性、独立、同方差、正态残差
- **多元线性回归**：Y = β₀ + β₁X₁ + … + βₖXₖ + ε；控制多个解释变量
- **逻辑回归**：P(Y=1|X) = 1/(1+e^(−(β₀+β₁X)))；二分类问题，用最大似然估计，输出概率而非类别
- **模型选择**：AIC = −2lnL + 2k（偏重拟合，适合预测），BIC = −2lnL + k·ln(n)（偏重简约，适合解释），调整 R² = 1 − (1−R²)(n−1)/(n−k−1)
- **偏差-方差权衡**：总误差 = 偏差² + 方差 + σ²（不可约误差）；交叉验证选择最优复杂度

### 第六步：实验设计 / Experimental Design

- **随机化 Randomization**：随机分配处理，消除系统偏差；使处理组与对照组在所有变量（已知与未知）上期望相等
- **区组化 Blocking**：将相似单元分组（如同年龄段、同地区），区组内比较减少混杂变异
- **样本量与功效分析 Power Analysis**：功效 Power = 1 − β = P(拒绝 H₀ | H₁ 为真)；给定效应量 δ、显著性水平 α，计算所需最小样本量 n；低功效 = 高假阴性风险
- **因子设计 Factorial Designs**：2ᵏ 设计同时研究 k 个因素及交互效应；比逐因素实验更高效，能发现交互作用

### 第七步：因果推理 / Causal Inference

- **混淆变量 Confounders**：同时影响处理 X 和结果 Y 的变量 Z，使 P(Y|X) ≠ P(Y|do(X))；不调整混淆则效应估计有偏
- **DAG 有向无环图**：可视化因果结构，节点=变量，箭头=直接因果；识别混淆路径（后门路径）、中介路径
- **do-演算 do-calculus**：Pearl 的干预算子，P(Y|do(X)) 表示"强制设定 X 时 Y 的概率"，区别于观察性 P(Y|X)
- **后门准则 Back-door Criterion**：若存在变量集 S 使得：(1) S 阻断 X→Y 的所有后门路径，(2) S 不包含 X 的后代，则调整 S 可得到因果效应 P(Y|do(X)) = Σₛ P(Y|X,S=s)·P(S=s)
- **随机化作为黄金标准**：随机实验直接阻断所有后门路径，无需识别混淆变量；无法随机化时依赖 DAG + 后门准则

### 第八步：解释结果 / Interpret Results

- 区分**统计显著**与**实际显著**（p < 0.05 不等于效应大；大样本下微小差异也能"显著"）
- **效应量 Effect Size**：Cohen's d = (μ₁−μ₂)/σ（标准化均值差）；odds ratio = (P₁/(1−P₁))/(P₂/(1−P₂))；不报告效应量的显著性检验是无意义的
- 注意**相关不等于因果**——关联 P(Y|X) ≠ 因果 P(Y|do(X))，除非消除了混淆
- 考虑**多重比较问题**——m 个独立检验，期望假阳性 = m·α；需 Bonferroni 或 BH-FDR 校正

### 第九步：量化不确定性 / Quantify Uncertainty

- 给出估计的**置信区间**（频率学派：重复抽样覆盖率）或**可信区间**（贝叶斯：参数在此区间的概率）
- 区分两种区间解释：频率学派 CI = "如果重复实验，95% 的区间包含真值"；贝叶斯 CrI = "参数在 此区间的概率为 95%"
- 说明结论的可靠性程度和统计功效
- 识别剩余的不确定性来源（模型假设、测量误差、未观测混淆）
- 报告效应量的置信区间，而非仅报告点估计

## 常见错误 / Common Errors

| 错误 / Error | 数学批评 / Mathematical Critique | 正确做法 / Correct Approach |
|---|---|---|
| 把 p-value 当真理 | p = 0.05 不表示"有 95% 的把握"，而是在 H₀ 下观察到该数据或更极端的概率 | 理解 p-value 的真正含义：P(数据或更极端 | H₀) |
| 混淆概率与频率 | 概率是理论值 P(A)，频率是经验值 f(A)/n；两者仅在大数定律下收敛 | 用大数定律连接两者：n→∞ 时 f/n → P |
| 忽视先验概率 | 基础率忽视：只看新证据，不看基础概率；罕见病检验中尤其危险 | 贝叶斯公式：P(H|E) = P(E|H)·P(H)/P(E) |
| 幸存者偏差 | 只看到成功者，没看到失败者；样本空间被截断 | 考虑完整样本空间 Ω，包括失败样本 |
| 小样本陷阱 | n = 3 的"规律"不可信；小样本方差估计不稳定 | 检查样本量，计算统计功效，用 t 分布而非正态 |
| 相关当因果 | 两个变量相关 ≠ 一个导致另一个；P(Y|X) ≠ P(Y|do(X)) | 寻找混淆变量，用 DAG + 后门准则做因果推断 |
| 赌徒谬误 | "已经连续 5 次正面了，下次该反面了"——每次独立事件 | 区分独立事件 P(第6次=H)=0.5 与依赖事件 |
| 忽视回归均值 | 极端表现后倾向于回归平均水平 E[X] | 用回归分析而非直觉判断趋势 |
| 小样本用正态而非 t 分布 | 当 n < 30 时，t 分布尾部更厚，正态近似低估不确定性；t 的置信区间更宽 | n < 30 必须用 t 分布，df = n − 1；n ≥ 30 时 t ≈ 正态 |
| 混淆 AIC 与 BIC | AIC 惩罚 2k 偏重拟合，BIC 惩罚 k·ln(n) 偏重简约；n 大时 BIC 选更简模型 | 目标为预测用 AIC，目标为解释用 BIC；两者矛盾时需进一步分析 |
| 观测数据推断因果 | 无随机化时，混淆变量无法消除；P(Y|X) ≠ P(Y|do(X))，效应估计有偏 | 用 DAG + 后门准则调整混淆集，或承认仅能推断关联而非因果 |
| 忽略多重比较校正 | m 个独立检验，期望假阳性 = m·α；α=0.05, m=20 → 期望 1 个假阳性 | Bonferroni：α_adj = α/m；BH-FDR 控制假发现率，更温和 |
| 忽略统计 power 与效应量 | 低 power 实验：即使真实效应存在也可能不显著；只报告 p 不报告效应量 | 设计前计算所需样本量，报告效应量 Cohen's d 或 odds ratio |

## 操作规程 / Operating Procedure

当本 skill 被触发时，执行以下具体步骤：

1. **随机变量定义**：明确 `[变量]: [含义] [类型: 离散/连续]`，指定样本空间 Ω 和事件域 𝔽
2. **概率模型**：说明使用的分布及理由 `[分布]: [选择] 因为 [理由]`，注明自由度 df 或关键参数
3. **数据评估**：`[样本量]: N, [偏差检查]: [结果], [质量]: [评估], [收集方式]: [随机/方便]`
4. **统计推断结果**：`[估计值]: [θ̂], [置信区间]: [a, b], [p-value]: [值], [估计方法]: [MLE/Bayesian]`
5. **回归建模（如适用）**：`[模型]: [线性/逻辑], [系数]: [β̂], [选择准则]: [AIC/BIC值], [R²]: [值]`
6. **实验设计评估（如适用）**：`[设计]: [随机化/区组/因子], [功效]: [Power值], [最小样本量]: [n]`
7. **因果评估（如适用）**：`[DAG]: [路径图], [混淆变量]: [列表], [调整集]: [S], [因果效应]: [P(Y|do(X))]`
8. **偏差检查**：逐一检查常见统计谬误（表中所列 13 项），标注 `✅ 无此问题` 或 `⚠️ 发现 [具体问题]`
9. **效应量与不确定性**：报告效应量 Cohen's d / odds ratio 及其置信区间，注明统计功效和剩余不确定性

**输出必须包含以上 9 项，不得只输出分析性文字而不给出结论。**

## 与其他 skill 的关系 / Relations to Other Skills

- **建模思想**：概率模型是建模的重要工具——不确定性需要概率框架
- **优化思想**：在不确定性下的优化需要期望值优化或鲁棒优化
- **归纳与类比**：从数据中发现规律本质上是归纳推理的概率版本
- **逻辑演绎**：经典逻辑是确定性的，概率逻辑是不确定性的推广
- **因果推断思想**：因果与统计的根本区分——P(Y|X) 是统计关联，P(Y|do(X)) 是因果效应；因果需要机制假设（DAG），不能纯靠数据得出；统计方法只能发现关联，因果推断思想提供从关联到因果的推理框架
- **信息论思想**：熵 H(X) = −ΣP(x)lnP(x) 量化随机变量的不确定性；信息增益 I(X;Y) = H(X) − H(X|Y) 衡量一个变量对另一个的减少不确定性能力；最大熵原则：在约束下选择使熵最大的分布（最少假设）
